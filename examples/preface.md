# Welcome to **mdpress**

mdpress is an open-source tool for converting Markdown-formatted books into high-quality PDFs.

## Why mdpress?

Markdown has become the de facto format for technical writing, but turning Markdown into a polished PDF is still tedious. mdpress exists to solve that problem.

Core advantages of mdpress:

- **High-quality typesetting**: Powered by Chromium rendering, with output quality close to professional publishing tools.
- **Easy to use**: One config file and a few Markdown files are enough to produce polished PDFs.
- **Highly customizable**: Customize themes, styles, fonts, and page layout.
- **Multi-chapter support**: Organize and build larger book projects with ease.

## Who is mdpress for?

- Technical book authors
- Open-source documentation maintainers
- Engineers who need PDF reports
- Anyone writing in Markdown who needs polished PDF output
