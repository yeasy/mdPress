# Preface

Welcome to **mdpress**.

mdpress is an open-source tool for publishing Markdown-formatted books as polished websites, PDFs, standalone HTML pages, and EPUB eBooks — all from the same source files.

## Why mdpress?

Markdown has become the de facto format for technical writing, but turning a pile of Markdown files into something you can ship — a documentation site, a printable PDF, an eBook — is still tedious. mdpress exists to solve that problem.

Core advantages of mdpress:

- **One source, many formats**: Build a static website, PDF, standalone HTML, and EPUB from the same Markdown files.
- **High-quality typesetting**: PDF output is powered by Chromium rendering, with quality close to professional publishing tools.
- **Easy to use**: One config file and a few Markdown files are enough to publish a book.
- **Highly customizable**: Choose built-in themes, and fine-tune the look with custom CSS.
- **Multi-chapter support**: Organize and build larger book projects with ease.

## Who is mdpress for?

- Technical book authors
- Open-source documentation maintainers
- Engineers who need polished reports or handbooks
- Anyone writing in Markdown who wants publication-ready output
