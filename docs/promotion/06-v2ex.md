# V2EX

## 标题

mdpress：Go 写的 Markdown 书籍发布工具，单二进制文件生成 PDF/HTML/ePub

## 正文

做了一个命令行工具 [mdpress](https://github.com/yeasy/mdpress)，用来把 Markdown 文件转换成 PDF、HTML、ePub 和静态网站。

### 解决什么问题

我维护了几本用 Markdown 写的开源技术书籍。GitBook 商业化之后开源版停更了，HonKit 也基本没人维护。mdBook 不支持 PDF 输出，Pandoc 生成好看的 PDF 需要装 LaTeX。

我的需求很简单：一个工具，读取现有的 Markdown 文件，输出 PDF + HTML + ePub，不要让我装 Node.js 或 LaTeX。

### 特点

- **单二进制文件**，Go 编写，`brew install mdpress` 搞定
- **零配置模式**：指向一个 .md 文件夹就能自动发现章节并构建
- **兼容 SUMMARY.md**：从 GitBook/HonKit 迁移无需改动
- **多格式输出**：`mdpress build --format pdf,html,site,epub` 一次生成全部
- **实时预览**：`mdpress serve` 启动本地服务器，保存后浏览器自动刷新
- **静态网站输出**：三栏 GitBook 风格，支持暗色模式、代码复制、全文搜索
- **中文支持良好**：Goldmark 解析 + Chromium 渲染，CJK 排版没有问题

### 安装

```bash
brew tap yeasy/tap && brew install mdpress
# 或者
go install github.com/yeasy/mdpress@latest
```

### 快速体验

```bash
mdpress quickstart mybook
mdpress serve mybook        # 浏览器打开 localhost:3456
mdpress build mybook --format pdf,html
```

### 技术栈

Go + Goldmark（Markdown 解析）+ chromedp（PDF 生成）+ Cobra（CLI）。PDF 输出需要系统有 Chrome/Chromium，HTML 和 ePub 输出不需要任何外部依赖。
