# mdPress 用户手册

> 现代化 Markdown 出版，从源文件到完成书籍。

mdPress 是一个命令行工具，将 Markdown 文件转换为精美排版的书籍。它可以生成多页网站、PDF、单页 HTML 和 ePub，全部源自同一份源文件。

## 为什么选择 mdPress？

写作应该很简单。你只需专注于 Markdown 内容，mdPress 处理其余所有事务 —— 排版、导航、搜索、语法高亮、数学公式、图表等。

mdPress 开箱即用，无需配置。只需指向一个 Markdown 文件目录，它就能构建完整的书籍。需要更多控制时，一个简单的 `book.yaml` 配置文件就能让你完全掌握结构、样式和输出。

## 主要特性

- **多种输出格式** —— 网站、PDF、HTML、ePub 和 Typst 一键生成
- **零配置模式** —— 自动发现和排序 Markdown 文件
- **实时预览** —— 热重载开发服务器
- **全文搜索** —— 内置客户端搜索功能
- **深色模式** —— 自动浅色/深色主题，支持系统检测
- **数学与图表** —— KaTeX 数学公式、Mermaid 和 PlantUML 图表
- **CJK 支持** —— 优先支持中文、日文和韩文排版
- **插件系统** —— 通过自定义钩子扩展构建流程
- **GitBook 迁移** —— 一键从 GitBook/HonKit 迁移
- **多语言支持** —— 用多种语言构建同一本书
- **增量构建** —— 基于哈希的缓存，快速重新构建

## 快速示例

```bash
# 安装
go install github.com/yeasy/mdpress@latest

# 创建新项目
mdpress quickstart my-book
cd my-book

# 在浏览器中预览
mdpress serve --open

# 构建所有格式
mdpress build --format all
```
